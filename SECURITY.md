# Security Policy

## Supported Versions

Security fixes are provided for the latest stable release line.

| Version | Supported |
| ------- | --------- |
| 0.2.x   | ✅        |
| < 0.2.0 | ❌        |

## Reporting a Vulnerability

Please **do not** report security vulnerabilities in public GitHub issues.

Instead, email: **shubhampardule@gmail.com**

If possible, include:

- Affected commit/tag/branch
- Clear reproduction steps
- Expected vs actual behavior
- Impact assessment
- Proof-of-concept (if available)

## Response Targets

- Acknowledge report within **48 hours**
- Confirm triage status as soon as practical
- Work on a fix and release when validated

Thank you for helping keep TransmitFlow safe.
